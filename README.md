# Selenium Automation Project

Proyecto de automatización funcional desarrollado con:

- Java 17
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)

## Casos automatizados
- Login exitoso
- Login inválido (usando DataProvider)

## Cómo ejecutar

1. Clonar el repositorio
2. Abrir en IntelliJ
3. Ejecutar desde Maven → Lifecycle → test
   o
   Ejecutar con:
   mvn clean test
